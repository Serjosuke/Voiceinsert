# Для работы требуется vosk-model-small-ru-0.22 в корневой папке

ГОЛОСОВОЕ ЗАПОЛНЕНИЕ ДОКУМЕНТА

1.Запустите программу
2.Выберите файл Word (.docx)
3.Нажмите Запуск или F9

КАК ДИКТОВАТЬ

Формат команды:

"поле значение"

Примеры:
• «фамилия иванов»
• «возраст сорок пять»
• «жалобы боль в груди»

МЕДИЦИНСКИЙ ДИКТАНТ
Режим чтобы лишние слова при записи не учитывались
Включите галочку «Медицинский диктант» или нажмите F8

В этом режиме говорить так:

запись поле значение

Пример:
• «запись давление сто двадцать на восемьдесят»

УПРАВЛЕНИЕ

F9 — Запуск
F10 — Стоп
F8 — Диктант ВКЛ / ВЫКЛ
Esc — Экстренная остановка

Голосом: «стоп»

РЕКОМЕНДАЦИИ

• Говорите чётко
• Делайте паузы
• Используйте одинаковые названия полей
